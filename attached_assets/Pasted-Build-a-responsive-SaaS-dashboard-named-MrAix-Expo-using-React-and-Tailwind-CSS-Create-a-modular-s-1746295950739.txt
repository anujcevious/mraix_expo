🧩 Layout Behavior
Header (Header.jsx)

Fixed top, full width

Left:

Round avatar with first letter of company name in theme color

Dropdown: Company name selector

Search bar

Right:

Dropdown: Period selector

Settings icon

Notification icon

User profile dropdown

Sidebar (Sidebar.jsx)

Fixed left side

Expandable: Shows full labels when expanded

Collapsed: Shows icons only, with hover text

Toggle via button

Uses vertical scrolling if too many items

ContentWrapper wraps everything except the header and sidebar

📱 Responsive Rules
On mobile (< md):

Table view (Table.jsx) is hidden using hidden md:block

CardView.jsx is shown with stacked info for each row (block md:hidden)

Make sure to test mobile layout with scroll, padding, and card tap interactions
